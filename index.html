What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a webpage as a tree of objects, allowing scripts (like JavaScript) to access, modify, and update the content, structure, and styling of a website dynamically.

How do you select an element from the DOM?

You can select an element from the DOM using methods like:

1. `document.getElementById("id")` for selecting by ID.

2. `document.querySelector(".class")` for selecting by class, tag, or any CSS selector.

3. `document.querySelectorAll(".class")` for selecting all matching elements.

What is the event delegation in the context of them, and is it useful?

Event delegation is a technique in JavaScript where a single event listener is added to a parent element to handle events from its child elements. Instead of adding multiple listeners to each child, the parent listens for events bubbling up from the children.


This is useful because it improves performance by reducing the number of event listeners, and it can handle dynamically added child elements without needing to reattach listeners.

How do you manipulate an element's attributes and styles using the dom?

To manipulate an element's attributes and styles using the DOM:


1. Attributes: Set an attribute: `element.setAttribute('attributeName', 'value');` - Get an attribute: `element.getAttribute('attributeName');`


2. Styles: Set a style: `element.style.propertyName = 'value';` Example: `element.style.color = 'red';`


3. Class Manipulation: Add a class: `element.classList.add('className');` - Remove a class: `element.classList.remove('className');`


This allows you to dynamically change element attributes, styles, and classes in your web application.
